# BentoKaze

BentoKaze is a linear programming calculator designed to minimize bento cost while ensuring nutritional requirements are met.

## Installation

Ensure you have Python 3.11 installed. Use Poetry to manage dependencies:

```bash
poetry install
```

## Dependencies

- Python 3.11
- PuLP 2.8.0
- Streamlit 1.35.0
- Pandas 2.2.2
